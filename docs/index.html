ğŸš€ Quick Start

Installation

Clone the repository and install dependencies:

git clone https://github.com/ax8tje/snaplet-pjatk-project.git
cd snaplet-pjatk-project
npm install

Platform-Specific Setup

  • Windows 11: See WINDOWS_SETUP.md
  • macOS/Linux: Follow the README instructions
  • iOS: Run cd ios && pod install

Run the App

# Start Metro bundler
npm start

# Run on Android
npm run android

# Run on iOS
npm run ios
